# 🔍 Бесплатный Zoom-виджет для Xbox Game Bar

> 🇬🇧 [Read in English](README.md)

Бесплатный open-source экранный увеличитель и прицел-оверлей для Windows в виде виджета Xbox Game Bar. Полная замена платного «Crosshair Zoom» — без ограничений по времени, без лицензий.

![Windows 10/11](https://img.shields.io/badge/Windows-10%20%7C%2011-blue) ![License: MIT](https://img.shields.io/badge/License-MIT-green)

## Возможности

| Функция | Описание |
|---------|----------|
| **Экранная лупа** | Захватывает монитор и рендерит увеличенное изображение в виджете |
| **Следование за курсором** | Зум следует за мышью в реальном времени (Lock выключен) |
| **Зум центра экрана** | Зум зафиксирован на центре монитора — виджет можно двигать куда угодно |
| **Зум произвольной точки** | Зафиксировать зум на любой точке экрана (Lock here) |
| **Прицел-оверлей** | 7 пресетов: Точка, Крест, Крест с зазором, Круг, Круг+Точка, Т-образный, Шеврон |
| **Настройка прицела** | Цвет (6 пресетов), размер, толщина, зазор |
| **Круглый / прямоугольный** | Обрезка зума по кругу или прямоугольнику |
| **Уровни увеличения** | от 1× до 8× |
| **Горячая клавиша** | Режим Hold или Toggle, любая клавиша, работает когда игра в фокусе |
| **Адаптивная производительность** | Автоматический пропуск кадров при нагрузке |
| **Изменение размера** | Слайдер в настройках для ресайза окна виджета |
| **Полная прозрачность** | Без рамок, без затемнения — только увеличенная картинка |
| **Закреплённый оверлей** | Закрепи виджет поверх игры, работает в pinned-режиме |

## Требования

- **Windows 10** версии 1809 (сборка 17763) или новее. Рекомендуется Windows 11.
- **Visual Studio 2022** с компонентом **Universal Windows Platform development**.
- **Режим разработчика** включён: Параметры → Для разработчиков.

## Сборка и установка

```powershell
# Клонируем
git clone https://github.com/Da3m0nRU/zoom-gamebar-widget.git
cd zoom-gamebar-widget/GameBarUWP

# Открываем в Visual Studio 2022
start GameBarWidget.sln
```

1. Платформа: **x64**.
2. **Сборка → Развернуть решение** (Build → Deploy Solution).
3. Открой Xbox Game Bar: `Win+G` → Меню виджетов → **Zoom**.

### Первый запуск

1. **Закрепи виджет** (кнопка-булавка 📌 в заголовке). Это обязательно для работы выбора экрана.
2. Закрой Game Bar (`Esc` или `Win+G`).
3. Нажми **Select screen** в виджете и выбери свой монитор.
4. Готово — виджет увеличивает изображение.

## Как это работает

```
┌──────────────────────────────────────────────────┐
│  Windows.Graphics.Capture (захват монитора)      │
│  ↓                                               │
│  CanvasBitmap → маска области виджета → выборка  │
│  ↓                                               │
│  Win2D CanvasSwapChain → рисуем увеличенное      │
│  ↓                                               │
│  Рисуем прицел поверх                            │
│  ↓                                               │
│  Present в окно виджета Game Bar                 │
└──────────────────────────────────────────────────┘
```

- **Без рекурсии**: область виджета закрашивается чёрным в захваченном кадре до семплирования.
- **Горячая клавиша**: `GetAsyncKeyState` + capability `inputForegroundObservation` — читает клавиши даже когда игра в фокусе.
- **Настройки**: хранятся в `ApplicationData.LocalSettings`, общие между виджетом зума и виджетом настроек.

## Настройки

Открываются по кнопке ⚙ в заголовке виджета.

### Раздел Zoom
- **Magnification** — от 1× до 8×
- **Hotkey** — привязка клавиши, режим Hold или Toggle
- **Lock Zoom Area** — Lock center (центр экрана) / Lock here (точка курсора) / Off (следовать за курсором)
- **Circular Zoom** — круг или прямоугольник
- **Adaptive Performance** — автоматический пропуск кадров
- **Frame Skip** — ручной 0–10

### Раздел Crosshair
- **Включить** — вкл/выкл
- **Стиль** — Точка, Крест, Крест+зазор, Круг, Круг+Точка, Т-образный, Шеврон
- **Цвет** — Зелёный, Красный, Голубой, Жёлтый, Белый, Пурпурный
- **Размер** — 4–200 пикселей
- **Толщина** — 1–10 пикселей
- **Зазор** — 0–50 пикселей

### Раздел Widget Window
- **Размер виджета** — слайдер (200–1000 px)
- **Центрировать окно** — поставить виджет в центр экрана

## Известные ограничения

- **Выбор экрана на Windows 10.** Система показывает одноразовый диалог выбора. На Windows 11 22H2+ можно обойтись без него.
- **Жёлтая рамка захвата на Windows 10.** Системная рамка вокруг захваченного монитора. На Windows 11 убирается через `IsBorderRequired = false`.
- **Виджет должен быть закреплён** перед нажатием Select screen. В режиме оверлея Game Bar блокирует системный picker.
- **Один монитор.** Виджет захватывает монитор, выбранный при пике. Для переключения нужно выбрать заново.

## Публикация

### GitHub (рекомендуется для сообщества)
Форкай, улучшай, делись. Лицензия MIT — делай что хочешь.

### Microsoft Store / Game Bar Widget Store
Можно опубликовать в Store:
- `inputForegroundObservation` и `graphicsCaptureProgrammatic` — restricted capabilities, Microsoft попросит обоснование. «Экранная лупа для доступности» — валидная причина.
- Убери `AppListEntry="none"` из манифеста, если нужна иконка в меню Пуск.
- Подпиши реальным сертификатом.
- Виджет автоматически появится в Widget Store внутри Game Bar.

## Структура проекта

```
App.xaml[.cs]              Маршрутизация активации виджетов
MainPage.xaml[.cs]         Рендер зума + отрисовка прицела
SettingsPage.xaml[.cs]     UI настроек
ErrorPage.xaml[.cs]        Страница ошибок (fallback)
CaptureHelper.cs           Interop (HWND, WDA, монитор, курсор, захват)
ZoomSettings.cs            Настройки (LocalSettings)
Package.appxmanifest       Два виджета (Standard + Settings)
```
