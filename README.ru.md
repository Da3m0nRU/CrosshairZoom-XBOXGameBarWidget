# 🔍 Бесплатный Zoom-виджет для Xbox Game Bar

> 🇬🇧 [Read in English](README.md)

Бесплатный open-source экранный увеличитель и прицел для Windows. Работает как виджет Xbox Game Bar — поверх любой полноэкранной игры. Без ограничений по времени, без лицензий, без оплаты — в отличие от платных аналогов.

![Windows 10/11](https://img.shields.io/badge/Windows-10%20%7C%2011-blue) ![License: MIT](https://img.shields.io/badge/License-MIT-green)

## Что умеет

- **Увеличивает** часть экрана (1×–8×) и показывает в закреплённом виджете поверх игры.
- **Следует за курсором** или фиксируется на точке (например, центр экрана — удобно для снайперки).
- **Рисует прицел** поверх увеличенного изображения — 7 форм, настраиваемый цвет/размер/толщина.
- **Работает в полноэкранных играх** — виджет является частью Xbox Game Bar, который рендерится поверх DirectX/Vulkan.
- **Горячая клавиша** — зажми или переключи клавишу, чтобы зум работал только когда нужен.

## Быстрый старт

### Вариант 1: Скачать релиз
Перейди в [Releases](../../releases), скачай `.appx` пакет и установи (нужен включённый Режим разработчика в Параметрах Windows).

### Вариант 2: Собрать из исходников

Требования:
- Windows 10 (1809+) или Windows 11
- Visual Studio 2022 с компонентом **Universal Windows Platform development**
- Включённый Режим разработчика

```
git clone https://github.com/user/zoom-gamebar-widget.git
cd zoom-gamebar-widget/GameBarUWP
```

Открой `GameBarWidget.sln` в Visual Studio → платформа **x64** → **Сборка → Развернуть решение**.

### Первый запуск

1. Нажми `Win+G` чтобы открыть Xbox Game Bar.
2. Найди **Zoom** в меню виджетов и открой.
3. **Закрепи виджет** (📌 в заголовке) — это обязательно.
4. Закрой Game Bar (`Esc`).
5. Нажми **Select screen** в виджете и выбери свой монитор.
6. Готово. Виджет увеличивает изображение.

## Настройки

Нажми ⚙ в заголовке виджета.

**Zoom**
| Настройка | Что делает |
|-----------|-----------|
| Magnification | Уровень увеличения (1×–8×) |
| Hotkey | Клавиша активации зума (режим Hold или Toggle) |
| Lock Zoom Area | Off = следовать за курсором. Lock center = всегда показывать центр экрана. Lock here = зафиксировать на текущей позиции курсора |
| Circular Zoom | Круглая или прямоугольная обрезка |
| Adaptive Performance | Автоматический пропуск кадров при нагрузке |
| Frame Skip | Ручной пропуск кадров (0–10) |

**Crosshair (Прицел)**
| Настройка | Что делает |
|-----------|-----------|
| Enable | Показать/скрыть прицел |
| Style | Точка, Крест, Крест с зазором, Круг, Круг+Точка, Т-образный, Шеврон |
| Color | Зелёный, Красный, Голубой, Жёлтый, Белый, Пурпурный |
| Size | Радиус прицела (4–200 px) |
| Thickness | Толщина линий (1–10 px) |
| Gap | Зазор в центре для стилей Plus/CircleDot (0–50 px) |

**Widget Window**
| Настройка | Что делает |
|-----------|-----------|
| Widget Size | Изменить размер окна виджета |
| Centralize window | Переместить виджет в центр экрана |

## Как это работает (для разработчиков)

- Захват экрана через `Windows.Graphics.Capture`.
- Рендеринг через Win2D (`CanvasSwapChain`).
- Виджет маскирует свою область в захваченном кадре, чтобы не было бесконечной рекурсии.
- Горячая клавиша через `GetAsyncKeyState` + capability `inputForegroundObservation`.
- Настройки между виджетами через `ApplicationData.LocalSettings`.

## Известные ограничения

- **Виджет нужно закрепить** перед выбором экрана. В режиме оверлея Game Bar блокирует системный picker.
- **Жёлтая рамка захвата на Windows 10** — системный индикатор, убрать нельзя. На Windows 11 отсутствует.
- **Один монитор** — захватывает выбранный при настройке. Для смены нужно выбрать заново.
